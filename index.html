📤 Buffer + GitHub

ℹ️ Buffer bloquea llamadas directas desde el browser. El bot usa GitHub Actions como intermediario seguro. Solo necesitas un GitHub Token con permiso workflow.
Crea uno en: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → scope: workflow
Primero agrega BUFFER_TOKEN y BUFFER_PROFILE_ID en GitHub → Settings → Secrets → Actions
